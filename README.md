# Description

The mobile app for Heroes 3 HOTA.
Consume the API https://github.com/proyectosbeta/api-heroes-3-HOTA

# Tecnologies

- React Native 0.64.xx
- Yarn 1.22
- OpenJDK 14
- Gradle 6.5

# Development

One terminal:

```
react-native start
```

Another terminal:

```
cd android && ./gradlew clean && cd ../
npx react-native run-android
```
