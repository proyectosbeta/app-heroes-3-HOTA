# Description

The mobile app for Heroes 3 HOTA.
Consume the API https://github.com/proyectosbeta/api-heroes-3-HOTA

# Tecnologies

- react native
- yarn
- open JDK 14
- gradle 6.5

# Development

```
react-native start
```

```
cd android && ./gradlew clean && cd ../
npx react-native run-android
```
